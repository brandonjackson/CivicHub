== Description ==
This plugin allows you to list the terms of any taxonomy in the form of a widget.
Supports several options to select, order, and display the terms.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'taxonomy-terms-widget' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Drag the widget to the appropriate sidebar from Appearance -> Widgets, and configure the options.
